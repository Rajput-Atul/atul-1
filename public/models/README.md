# 3D Models Directory

## Required Models

Download free models from these sources and place them here:

### Spaceship
- Source: Sketchfab, Poly Pizza, Kenney Assets
- Format: GLB/GLTF
- Size: Keep under 5MB
- Place in: `public/models/spaceship/`

### Robots/Drones
- Source: Sketchfab, Quaternius
- Format: GLB/GLTF
- Place in: `public/models/robots/`

### Asteroids
- Source: Sketchfab, OpenGameArt
- Format: GLB/GLTF
- Place in: `public/models/asteroids/`

### Satellites
- Source: Sketchfab
- Format: GLB/GLTF
- Place in: `public/models/satellites/`

## Optimization
- Use Draco compression
- Reduce polygon count
- Use KTX2 textures
- LOD for distant objects
