# HDR Environment Maps

## Required HDRIs

Download free HDR environment maps from Poly Haven:
- https://polyhaven.com/hdris

## Needed HDRIs

### Space Environments
- `space_01.hdr` — Deep space starfield
- `nebula_01.hdr` — Colorful nebula background
- `dock_01.hdr` — Industrial hangar lighting

## Usage
- Used for environment lighting in Three.js
- Load with `RGBELoader`
- Apply to scene background or environment map

## Optimization
- Use 1K or 2K resolution
- Convert to `.hdr` format
- Compress with texture tools if needed
