# Audio Directory

## Required Sound Effects

Download free audio from these sources:
- **Pixabay**: https://pixabay.com/sound-effects/
- **Mixkit**: https://mixkit.co/free-sound-effects/
- **Freesound**: https://freesound.org/ (check licenses)
- **OpenGameArt**: https://opengameart.org/

## Needed Sounds

### Ambient
- `space_ambient.mp3` — Deep space background hum
- `engine_idle.mp3` — Low engine vibration
- `reactor_hum.mp3` — Reactor pulse sound

### UI
- `hover.mp3` — Button hover
- `click.mp3` — Button click
- `notification.mp3` — Alert/achievement

### Hologram
- `hologram_activate.mp3` — Hologram appearing
- `hologram_deactivate.mp3` — Hologram fading

### Doors
- `door_open.mp3` — Airlock/hatch opening
- `door_close.mp3` — Airlock/hatch closing

### Alert
- `alert_soft.mp3` — Soft notification
- `alert_critical.mp3` — Critical warning

## Format
- MP3 (compressed)
- Keep under 500KB each
- Sample rate: 44.1kHz or 48kHz
